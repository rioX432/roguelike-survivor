# GlitchClaw Audio Files

Drop your audio files here with these exact names:

BGM:
  bgm_main.ogg        - Main game loop BGM (Celtic x Gagaku x J-Rock, BPM 150)

SE:
  se_attack_fire.ogg  - Player attack shot (energy blast)
  se_enemy_hit.ogg    - Enemy takes damage (metallic clank)
  se_enemy_die.ogg    - Enemy destroyed (crash + electronic glitch)
  se_level_up.ogg     - Level up! (sparkle chime)
  se_player_hit.ogg   - Player takes damage (soft hurt sound)

Supported formats: .ogg, .mp3, .wav
Unity will import automatically.
